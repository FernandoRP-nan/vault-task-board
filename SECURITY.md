# Política de seguridad

## Versiones soportadas

| Versión | Soportada |
|---------|-----------|
| 1.1.x   | Sí        |
| < 1.1   | No        |

## Reportar una vulnerabilidad

Si encuentras un problema de seguridad, abre un [issue privado](https://github.com/FernandoRP-nan/vault-task-board/security/advisories/new) o contacta al mantenedor vía GitHub.

No publiques exploits activos en issues públicos.

Este complemento procesa datos locales del vault; no realiza peticiones de red por defecto.
