# Contribuir

Gracias por tu interés en mejorar Task Board.

## Cómo empezar

1. Haz fork del repositorio.
2. Clona tu fork **fuera** de cualquier vault de Obsidian (evita indexar `node_modules`).
3. Instala dependencias: `npm install`
4. Compila: `npm run build` o `npm run dev` para desarrollo con watch.

## Pull requests

- Describe el problema o la mejora con claridad.
- Mantén los cambios acotados a un tema por PR.
- Verifica que `npm run build` termine sin errores.
- No incluyas datos personales ni archivos `.db` de prueba.

## Reportar bugs

Abre un issue con: versión del plugin, versión de Obsidian, sistema operativo y pasos para reproducir.
